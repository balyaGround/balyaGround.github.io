About Me

Haloo saya adalah seorang web developer.Nama saya Balya saya lahir di Medan 11 november 1996 untuk sekarang saya masih seorang freelance web front end. Saya lulus dari SMAN 18 Medan pada tahun 2014 kemudian karena keadaan yang tidak menguntungkan saya harus menerima kenyataan bahwa saya tidak dapat melanjutkan kuliah pada tahun 2014.Kemudian akhirnya pada tahun 2015 saya bisa dapat merasakan perkuliahan tepatnya saya kuliah di Usu pada jurusan yang juga saya sukai yaitu Ilmu Komputer. Di tahap ini saya banyak belajar tidak hanya mengembangkan Hard Skill namun juga meningkatkan Soft Skill saya dalam berinteraksi dengan orang banyak. Alhamdulillah saya berhasil lulus tepat waktu pada tahun 2019 dan sekarang saya sedang Bootcamp di Binar acamdemy agar saya dapat mendapatkan pekerjaan impian menjadi seorang web Fullstack Dev.
